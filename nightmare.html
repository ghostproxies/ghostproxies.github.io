public class Nightmare8 { public ulong a; public ulong b; public byte Next() { a += 1111111111; b += (b >> 8) + a; return (byte)b; } }

The Next function modifies the state in a Nightmare8 instance to generate a pseudorandom byte integer as the return value.

Each state variable (a and b) in each Nightmare8 instance must be seeded.
